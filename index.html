Desarrollo web front

El desarrollo web front-end se enfoca en la creación de la interfaz de usuario que los usuarios ven e interactúan en una página web. Esto incluye la estructura (HTML), los estilos (CSS) y la lógica interactiva (JavaScript) del sitio web.

HTML, CSS y JS

HTML es el lenguaje que estructura el contenido de una página web. CSS se utiliza para darle estilo, colores y diseño. JavaScript permite crear funciones interactivas, dinámicas y responder a acciones del usuario.
